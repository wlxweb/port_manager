# Security Policy

## Supported Versions

Security updates are provided for the latest released version.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report security concerns privately to the repository owner through GitHub. Include:

- A description of the issue.
- Steps to reproduce.
- The operating system and app version.
- Any relevant logs or screenshots.

## Scope

Port Manager runs local system commands to inspect ports and processes. Reports related to unsafe command handling, unintended process termination, privilege escalation, or exposure of sensitive local process data are in scope.
